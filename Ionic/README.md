Ionic 使用
===

1. npm install ionic cordova -g
>
    ionic -h
    cordova -h

2. ionic start projectName
>
    $ ionic start myApp blank --v2
    $ ionic start myApp tabs --v2
    $ ionic start myApp sidemenu --v2

3. npm install

4. ionic serve

打包环境
===
ionic platform add android
ionic build android
ionic emulate android
